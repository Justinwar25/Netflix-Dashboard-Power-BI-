# Netflix-Dashboard-Power-BI-
Interactive Power BI report that explores Netflix titles and trends (content type, genres, ratings, release year, and country distribution) with drilldowns and slicers for fast filtering.

This Power BI dashboard analyzes Netflix titles using audience ratings and streaming volume. It highlights overall performance at a glance and supports filtering and drill-down analysis by origin and content rating category.

Key visuals

KPI Card: Average Audience Rating of all shows/movies
Donut Chart: Average Monthly Streams by Origin (Domestic vs International)
Line Chart: Average Audience Rating by Release Year, segmented by Rating Description
Detail Table: Title-level breakdown with Average Monthly Streams and Average Audience Rating

Interactivity

Visuals cross-filter each other to explore trends and compare categories quickly.

Tools used

Power BI Desktop (Power Query + DAX measures)
